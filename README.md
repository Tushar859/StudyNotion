# StudyNotion - EdTech Platform

StudyNotion is a fully functional EdTech platform built using the MERN Stack.  
It provides a modern learning environment where students can enroll in courses and instructors can create and manage educational content.

---

## 🚀 Features

### 👨‍🎓 Student Features
- User Authentication & Authorization
- Browse Courses
- Purchase Courses
- Enroll in Courses
- Watch Course Videos
- Course Progress Tracking
- Profile Management

### 👨‍🏫 Instructor Features
- Create & Manage Courses
- Upload Course Content
- Add Sections & Subsections
- Instructor Dashboard
- Track Student Enrollments

### 🔐 Authentication
- JWT Authentication
- Login & Signup
- OTP Verification
- Forgot Password Functionality

### 💳 Payment Integration
- Secure Payment Gateway Integration

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Tushar859/StudyNotion.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd StudyNotion
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Start Frontend

```bash
npm start
```

### 5️⃣ Start Backend

```bash
cd server
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file in the server directory and add:

```env
MONGODB_URL=your_mongodb_url
JWT_SECRET=your_secret_key
CLOUDINARY_KEY=your_cloudinary_key
RAZORPAY_KEY=your_razorpay_key
```
